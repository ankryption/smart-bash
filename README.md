## Steps to customise

```bash

git clone https://github.com/mehtankush/my-terminal.git

# Go into the repository
cd my-terminal

# Move files starting with `.` to the `root` i.e. ~/
mv .bashrc .bash_profile .bash_aliases .git-prompt.sh ~/
```
